# SuperZen Project Rules
- **Aesthetic:** Dark mode, frosted glass (Vibrancy), rounded corners (min 12pt).
- **Core Loop:** 20-20-20 Rule.
- **System Level:** Must run in Menu Bar. No Dock icon.
- **Privacy:** All eye-tracking and meeting-detection data stays on-device.
- **Pop-up Logic:** Use a 60-second countdown nudge (as seen in screenshots) before mandatory full-screen breaks.
